Representacion gráfica para el problema 3D BIN PACKING  mediante  el algoritmo (http://www.diku.dk/~pisinger/codes.html ) . 
Optimiza el volumen de cubos rectangulares dentro de un contenedor rectangular
Muestra el resultado en un navegador web Chrome,  
Requiere un servidor web con PHP  y Visual studio C++ 2012



Graphical representation for the 3D BIN PACKING problem using the algorithm (http://www.diku.dk/~pisinger/codes.html).
Optimizes the volume of rectangular cubes inside a rectangular container
Displays the result in a Chrome web browser,
Requires a web server with PHP and Visual studio C ++ 2012
